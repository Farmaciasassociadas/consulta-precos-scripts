# Scripts Violentmonkey — Assistente de Preços EAN

Repositório **público só para os 4 scripts** (Droga Raia, Nissei, São João,
Farmácias São Paulo). Fica público de propósito, para o Violentmonkey
conseguir buscar atualizações direto daqui (o GitHub bloqueia acesso anônimo
ao conteúdo de repositórios privados).

**Não há dados de preço, EANs nem informação de negócio aqui** — isso vive no
repositório privado `Farmaciasassociadas/consulta-precos-drogaraia`, junto
com o programa Python (`assistente_eans.py`).

## Instalar (uma vez)
Abra cada link abaixo no Chrome com o Violentmonkey instalado e clique em
"Instalar":

- [captura_preco.user.js](https://raw.githubusercontent.com/Farmaciasassociadas/consulta-precos-scripts/main/captura_preco.user.js) — Droga Raia
- [captura_preco_nissei.user.js](https://raw.githubusercontent.com/Farmaciasassociadas/consulta-precos-scripts/main/captura_preco_nissei.user.js) — Nissei
- [captura_preco_saojoao.user.js](https://raw.githubusercontent.com/Farmaciasassociadas/consulta-precos-scripts/main/captura_preco_saojoao.user.js) — São João
- [captura_preco_saopaulo.user.js](https://raw.githubusercontent.com/Farmaciasassociadas/consulta-precos-scripts/main/captura_preco_saopaulo.user.js) — Farmácias São Paulo

Depois disso, o botão ⟳ "Verificar atualizações" do Violentmonkey busca a
versão mais nova direto daqui.
